About Me

안녕하세요! 생성형 AI를 활용하여 사용자 경험을 혁신하는 개발자 윤동준입니다.

저는 단순히 기술을 구현하는 것을 넘어, 사용자의 실제 문제를 해결하는 서비스를 만드는 것에 열정을 가지고 있습니다. "왜 이 기능이 필요한가?"를 항상 고민하며, 기술이 사람들의 일상에 자연스럽게 녹아들 수 있는 방법을 모색합니다.

특히 Gemini AI와 TMDB API를 활용한 MovieTalk 프로젝트에서는 "영화 선택의 피로감"이라는 일상적인 문제를 AI 대화를 통해 해결했습니다. 사용자가 자연어로 취향을 표현하면, AI가 이를 이해하고 최적의 영화를 추천하는 1인 개발 프로젝트를 완성했습니다.

또한 JSP 팀 프로젝트에서는 명일방주 게임 도우미 웹사이트를 개발하며 외부 API 연동, 데이터베이스 설계, 팀 협업 경험을 쌓았습니다.

동양미래대학교 컴퓨터소프트웨어공학과에서 웹 개발과 AI 기술을 공부하고 있으며, 배운 기술을 실제 프로젝트에 적용하며 성장하는 개발자가 되고자 합니다.

MovieTalk - AI 영화 추천 시스템

2025.11 ~ 현재 | 1인 개발

Tech Stack

Spring Boot Gemini AI TMDB API JavaScript HTML/CSS RESTful API

문제 정의

수많은 OTT와 영화 속에서 취향에 맞는 작품을 찾는 피로감, 단순 검색과 필터만으로는 미묘한 감성이나 분위기를 찾기 어려운 문제를 해결하고자 했습니다.

핵심 기능

AI 추천부터 시청까지

"인생 영화는 '인셉션'인데, 오늘은 혼자 볼 감성적인 영화 추천해줘!" 같은 자연어 입력을 분석, Gemini AI가 영화 추천 + 이유 + 시청 가능 OTT 정보를 한 번에 제공합니다.

대화로 성장하는 AI

"코미디 빼줘" 등 대화 맥락을 기억하고, 사용자 감상을 공유하며 취향을 학습합니다. "여운이 남는" 같은 추상적 표현도 분석하여 의도를 파악합니다.

나의 역할 (1인 개발)

  • 전체 프로젝트 기획, 사용자 문제 정의, 솔루션 도출
  • Spring Boot 기반 RESTful API 설계 및 백엔드 개발
  • Gemini AI 프롬프트 엔지니어링 및 자연어 처리 로직 개발
  • TMDB API 연동 (영화 데이터, OTT 정보 조회)
  • 대화형 프론트엔드 UI/UX 구현

배운 점

  • 생성형 AI (Gemini) 실전 활용
  • 외부 API (TMDB) 연동 노하우
  • 대화형 UX 설계 및 구현

개선 계획

  • 사용자 계정 시스템 (대화 히스토리)
  • 감상평 기반 추천 정확도 향상
  • OTT 가격 비교 기능 추가

명일방주 게임 도우미 웹사이트

2024.09 ~ 2024.12 | 팀 프로젝트

Tech Stack

JSP & Servlet JavaScript HTML/CSS MS-SQL MVC 패턴

프로젝트 개요

명일방주 게임 플레이어를 위한 종합 정보 및 계산 도구 웹사이트입니다. 게임 내 복잡한 계산을 자동화하고, 외부 API를 연동하여 실시간 게임 데이터를 제공합니다.

주요 기능

  • 공개모집 태그 조합 계산기 (최적 조합 추천)
  • 레벨링 및 재료 계산기 (필요 자원 자동 계산)
  • 스테이지별 아이템 드롭률 정보 제공 (외부 API 연동)
  • 회원 관리 시스템 및 커뮤니티 게시판

기술적 성취

  • MVC 패턴 기반 웹 애플리케이션 설계 및 구현
  • 데이터베이스 설계 및 최적화 (MS-SQL)
  • 외부 게임 데이터 API 연동 및 파싱
  • 팀 프로젝트 협업 경험 (Git 버전 관리)

About Me

안녕하세요! 생성형 AI를 활용하여 사용자 경험을 혁신하는 개발자 윤동준입니다.

저는 단순히 기술을 구현하는 것을 넘어, 사용자의 실제 문제를 해결하는 서비스를 만드는 것에 열정을 가지고 있습니다. "왜 이 기능이 필요한가?"를 항상 고민하며, 기술이 사람들의 일상에 자연스럽게 녹아들 수 있는 방법을 모색합니다.

특히 Gemini AI와 TMDB API를 활용한 MovieTalk 프로젝트에서는 "영화 선택의 피로감"이라는 일상적인 문제를 AI 대화를 통해 해결했습니다. 사용자가 자연어로 취향을 표현하면, AI가 이를 이해하고 최적의 영화를 추천하는 1인 개발 프로젝트를 완성했습니다.

또한 JSP 팀 프로젝트에서는 명일방주 게임 도우미 웹사이트를 개발하며 외부 API 연동, 데이터베이스 설계, 팀 협업 경험을 쌓았습니다.

동양미래대학교 컴퓨터소프트웨어공학과에서 웹 개발과 AI 기술을 공부하고 있으며, 배운 기술을 실제 프로젝트에 적용하며 성장하는 개발자가 되고자 합니다.

MovieTalk - AI 영화 추천 시스템

2024.11 ~ 현재 진행중 | 1인 개발
"우리의 영화 찾기는 왜 어려운가?"

수많은 OTT 서비스와 영화 속에서 내 취향에 딱 맞는 작품을 찾는 건 너무나 어려운 일입니다. 단순 검색과 필터만으로는 미묘한 감성과 분위기까지 찾아낼 수 없습니다.

MovieTalk은 자연어 대화를 통해 사용자의 취향을 이해하고, AI가 최적의 영화를 추천하는 대화형 영화 추천 시스템입니다.
Spring Boot Gemini AI (LLM API) TMDB API JavaScript HTML/CSS RESTful API

해결하고자 한 문제

  • 정보의 홍수: 어떤 영화를 봐야 할지 선택하기 위한 탐색 과정의 피로감
  • 취향 표현의 한계: "감성적", "긴장감 있는" 등 키워드로는 설명하기 힘든 복합적인 취향
  • 소통의 부재: 왜 이 영화가 추천되었는지 알 수 없는 일방적인 알고리즘

아키텍처 및 설계 결정

  • 외부 API 적극 활용: Gemini AI로 자연어 처리, TMDB API로 영화 데이터 확보
  • Spring Boot RESTful API: 확장 가능한 백엔드 아키텍처 구축
  • 대화형 UX: 영화 추천부터 시청까지 한 번에 해결하는 사용자 경험 설계

핵심 기능 1: AI 추천부터 시청까지 한 번에

  • 사용자 입력 예시: "인생 영화는 '인셉션'이랑 '어바웃 타임'인데, 오늘은 혼자 볼만한 감성적인 영화 추천해줘!"
  • AI가 영화 추천 + 추천 이유 + 시청 가능 OTT 정보를 한 번에 제공
  • TMDB API 연동으로 20,000개 이상의 영화 데이터베이스 활용
  • 실시간 OTT 가격 정보 제공 (예: 왓챠 월 7,900원~)

핵심 기능 2: 대화로 성장하는 AI 영화 메이트

  • 대화 맥락 기억: "코미디 빼줘", "방금 본 영화 감상" 등 후속 대화 처리
  • 자연스러운 피드백: 사용자 감상을 공유하며 AI와 취향 학습
  • 감성적 이해: "혼자 볼만한", "여운이 남는" 등 추상적 표현도 정확하게 분석
  • Gemini AI를 활용한 자연어 처리로 사용자 의도 파악

나의 역할 (1인 개발)

  • 전체 프로젝트 기획 및 설계: 사용자 문제 정의부터 솔루션 도출까지
  • Spring Boot 백엔드 개발: RESTful API 설계 및 구현
  • Gemini AI 연동: 프롬프트 엔지니어링 및 자연어 처리 로직 개발
  • TMDB API 연동: 영화 데이터 검색, 파싱, OTT 정보 조회
  • 프론트엔드 UI/UX 구현: 대화형 인터페이스 설계 및 개발

개발 방향 및 성과

  • 3단계 개발 완료: TMDB 연동 → LLM 대화 기능 → UI/UX 완성
  • 5단계 AI 추천 알고리즘: 분석 → 검색 → 필터링 → 선택 → 설명
  • 외부 API 통합 경험: 2개의 주요 API를 효과적으로 연동
  • 대화형 AI 서비스 구현: 단순 검색을 넘어 사용자와 소통하는 시스템 개발

배운 점

  • 생성형 AI의 실전 활용: Gemini AI를 활용한 자연어 처리와 프롬프트 엔지니어링
  • 외부 API 연동 노하우: 여러 API를 조합하여 시너지를 만드는 방법
  • 대화형 UX 설계: AI와 사용자 간 자연스러운 대화 흐름 구현
  • 문제 해결 중심 개발: 기술이 아닌 사용자 경험에서 시작하는 개발 프로세스

향후 개선 계획

  • 사용자 계정 시스템: 대화 히스토리 저장으로 더욱 개인화된 추천
  • 감상평 기반 학습: 사용자 리뷰를 분석하여 추천 정확도 향상
  • OTT 가격 비교: 여러 플랫폼의 가격을 비교하여 경제적인 시청 방법 제안
  • 추천 알고리즘 고도화: 더 많은 맥락 정보를 활용한 정교한 추천
